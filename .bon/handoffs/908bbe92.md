## Risks
- File-backed rooms stay in memory indefinitely (no eviction). Long-running server + many .md files = unbounded memory.
- `_safe_doc_path` uses `Path.resolve()` which follows symlinks — a symlink inside docs_dir pointing outside it bypasses the check.
- flush_doc's git commit uses `_find_git_root` walking up — if docs_dir is deep in a monorepo, the commit happens in the monorepo root, which may not be intended.
